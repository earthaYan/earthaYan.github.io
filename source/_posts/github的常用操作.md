issue 标签页：

- 查询所有状态为 open 的 issue
  `is:issue is:open`
- 查询所有状态为 open 的 PR
  `is:pr is:open`
- 查询指定发布者的 issue
  `is:issue author:_author_name_`
- 查询自己的 issue
  `is:issue author:@me`
- 查询指定标签的 issue
  `is:issue label:"_label_name_"`
- 查询没有指定标签的issue
`is:issue no:label`
