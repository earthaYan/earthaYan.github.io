### 问题描述
假设高度已知，写一个三栏布局，左右各200px,中间宽度自适应
<a name="DQ6JE"></a>

### 解决方案or思路

1. 浮动
2. 绝对定位
3. flex-box布局
4. 网格grid布局
5. 表格布局tab-cell的使用

<a name="UXJHb"></a>

### 具体代码
<font color=Red> 浮动定位</font>
```html
    <div class="container1">
        <div class="left">左边200</div>
        <div class="right">右边200</div>
        <div class="center">中间自适应</div>
    </div>
```

```less
.container1{
    div{
        height: 100px;
    }
    .left{
        background: red;
        width: 200px;
        float: left;;
    }
    .right{
        background: yellow;
        width: 200px;
        float: right;
    }
    .center{
        background: green;
    }
}
```

<font color=Red> 绝对定位</font>

```html
    <div class="container2">
        <div class="left">左边200绝对定位</div>
        <div class="center">中间自适应-绝对定位</div>
        <div class="right">右边200绝对定位</div>
    </div>
```

```less
.container2{
    position: relative;
    div{
        height: 100px;
        position: absolute;
    }
    .left{
        background: yellow;
        left: 0;

    }
    .right{
        background: red;
        right: 0;
        width: 200px;
    }
    .center{
        background: cyan;

        right: 200px;
    }
}
```

### 各个方法的优缺点
| 方法 | 优势 | 劣势 | 去掉高度已<br />知后的适用性 |
| --- | --- | --- | --- |
| 浮动 | 兼容性较好 | 需要清除浮动以及处理好浮动元素周围的<br />元素之间的关系 | 否 |
| 绝对定位 | 快捷不易出问题 | 所有的子元素会脱离文档流，可使用性较差 | 否 |
| flex布局 | 比较完美，尤其是在移动端更为方便 | 备注：css3属性 | 是 |
| 表格布局 | 兼容性好，可作为flex的补充（ie8） | 当某个单元格高度超出的时候，其他单元格会<br />自动增高 | 是 |
| 网格布局（新） | css开始支持，同任务代码更简单 |  | 兼容性差 |

### 拓展问题

<font color=Red>?</font>垂直三栏布局，两边定高，中间自适应（网格 flex  绝对定位目前可实现）
```html
    <div class="container">
        <div class="top">上200</div>
        <div class="middle">中间自适应</div>
        <div class="bottom">下200</div>
    </div>
```
<b>flexbox布局：当高度不足400的时候，定高会同时缩小</b>

```less
.container{
    height: 100%;
    display: flex;
    flex-direction: column;
    div{
        width: 100px;
      	min-height:100px;
    }
    .top{
        background: red;
        height: 200px;
    }
    .middle{
        background: orange;
        flex: 1;
    }
    .bottom{
        background: green;
        height: 200px;
    }
}
```

<b>绝对定位</b>

```less
.container{
    height: 100%;
    div{
        width: 100px;
      	position:absolute;
    }
    .top{
        background: red;
        height: 200px;
      	top:0;
    }
    .middle{
        background: orange;
        top:200px;
      	bottom:200px;
    }
    .bottom{
        background: green;
        height: 200px;
      	bottom:0
    }
}
```

<b>表格布局(在元素外部包裹一层显示为tab-row,真实元素显示tab-cell)</b>
```less
.container{
    height: 100%;
    display: flex;
    flex-direction: column;
    div{
        width: 100px;
    }
    .top{
        background: red;
        height: 200px;
    }
    .middle{
        background: orange;
        flex: 1;
    }
    .bottom{
        background: green;
        height: 200px;
    }
}
```

<b>网格布局</b>

```less
.container4{
    height: 100%;
    display: grid;
    grid-template-rows: 200px auto 200px;
    grid-template-columns: auto;
    div{
        width: 100px;
    }
    .top{
        background: red;
    }
    .middle{
        background: orange;
    }
    .bottom{
        background: green;
    }
}
```

<font color=Red>?</font>两栏布局

```html
    <div class="container">
        <div class="left">左邊300</div>
        <div class="right">右邊自適應</div>
    </div>
```
<b>浮动</b>

```less
.container{
    div{
        height: 200px;
    }
    .left{
        float: left;
        width: 300px;
        background: red;
    }
    .right{
        background: green;
    }
}
```

<b>flex布局</b>

```less
.container{
    display: flex;
    div{
        height: 200px;
    }
    .left{
        width: 300px;
        background: red;
    }
    .right{
        background: green;
        flex:1;
    }
}
```

<b>网格布局</b>

```less
.container{
    display: grid;
    grid-template-columns: 100px auto;
    grid-template-rows: 200px;
    div{
        // height: 200px;
    }
    .left{
        background: red;
    }
    .right{
        background: green;
    }
}
```

<b>绝对定位</b>

```less
.container{
    width: 100%;
    position: relative;
    div{
        height: 200px;
        position: absolute;
    }
    .left{
        background: red;
        left: 0;
        width: 200px;
    }
    .right{
        background: green;
        left: 200px;
        width: 100%;//或者设置right为0
    }
}
```

<b>表格布局</b>

```less
.container{
    width: 100%;//必须设置
    display: table;
    div{
        height: 100px;  
    }
    .left{
        background: red;
        width: 200px;
        display: table-cell;
    }
    .right{
        background: green;
        display: table-cell;
    }
}
```
